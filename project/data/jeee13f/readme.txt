Justice Expenditure and Employment Extracts 2013, Final		
This zip archive contains tables in individual  .csv spreadsheets		
from Justice Expenditure and Employment Extracts 2013, Final  NCJ 250327.  The full report including text		
and graphics in pdf format is available from: http://www.bjs.gov/index.cfm?ty=pbdetail&iid=5965		
		
This report is one in a series.  More recent editions may be available. To view a list of all in the series go to http://www.bjs.gov/index.cfm?ty=pbse&sid=33
